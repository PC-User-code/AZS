Проект сервер для АЗС
Цель проекта:
Разработка сервера по средствам билиотеки Flask, подключение к БД, 
обработка транзакцийЖ визуализация данных от станций, отображение состояния станции

Библиотеки(предполагаемые):
Flask
json, request и подобные(для обработки запросов и данных)
sqlite(подключение к БД, её изменение, добавление записей)
YandexMaps(Api) для отображения всех АЗС системы на карте
Различные HTML, CSS< JavaScript-скрипты для сайта
Внутренние библиотеки Python
